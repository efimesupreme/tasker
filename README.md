# Проектный трекер

Базовый каркас статического веб-приложения для публикации через GitHub Pages.

## Что внутри

- `index.html` — HTML-разметка интерфейса с боковой панелью, верхней строкой, поиском, кнопкой создания проекта и карточками-виджетами.
- `styles.css` — адаптивная тёмная тема без внешних CSS-библиотек.
- `app.js` — демо-массив проектов, рендеринг карточек, базовые CRUD-формы для проектов, групп и подзадач, локальное сохранение в браузере.

## Как открыть локально

Откройте файл `index.html` в браузере двойным кликом или запустите простой статический сервер из корня проекта:

```bash
python3 -m http.server 8000
```

После запуска сервера приложение будет доступно по адресу <http://localhost:8000>. Изменения в проектах сохраняются только в `localStorage` текущего браузера и не синхронизируются.

## Публикация на GitHub Pages

Проект не требует сборки. Для публикации достаточно разместить эти файлы в репозитории и включить GitHub Pages для нужной ветки в настройках репозитория.
